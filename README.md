Es una app de escritorio cuya funcion en consultar con el api de navixy para ver la lista de los clientes y sus trackers, permite consultar los clientes cuyos vehiculos no hacen realizado un update en un periodo de tiempo definido por el usuario. Al final de la consulta realiza un informe y convirte los datos en un archivo excel, el cual puede ser mandado por correo o almacenado de manera local.
