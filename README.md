Clienvix.

Desarrollado en python 3.12.3

App de escritorio, consulta con el api de navixy para ver la lista de los clientes, trackers y consultar las ultimas actualizaciones de los GPS en un periodo de tiempo definido por el usuario. luego realiza un informe y convirte los datos en un archivo excel, el cual puede ser almacenado de manera local o enviado por correo.
